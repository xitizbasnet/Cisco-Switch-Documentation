# 📘 Cisco VLAN Configuration Guide

### 🔧 VLAN Creation | Access Port | Trunk Port Configuration (Cisco 2960)

---

## 📌 Overview

Hello friends, welcome back. In this guide, I will show you:

- How to configure a VLAN in a Cisco switch
- How to assign a name to a VLAN
- How to assign a switch port to a VLAN (Access Port)
- Difference between Access Port and Trunk Port
- How to configure a Trunk Port in Cisco switch

📍 Device used: **Cisco 2960 Series Switch**

---

## 🔐 Accessing the Switch

```Shell
Switch> enable
Switch#
```

---

## 🔍 Step 1: Check Existing VLANs

```Shell
Switch# show vlan brief
```

### ✅ Default VLAN Information

- VLAN 1 → Default VLAN (Active)
- All ports are assigned to VLAN 1 by default

### 📦 Built-in VLANs

- VLAN 1002
- VLAN 1003
- VLAN 1004
- VLAN 1005

---

## 🆕 Step 2: Create VLANs and Assign Names

### 🔧 Enter Global Configuration Mode

```Shell
Switch# configure terminal
```

---

### ✅ Create VLAN 30 (Data)

```Shell
Switch(config)# vlan 30
Switch(config-vlan)# name DATA
Switch(config-vlan)# exit
```

---

### ✅ Create VLAN 50 (CCTV)

```Shell
Switch(config)# vlan 50
Switch(config-vlan)# name CCTV
Switch(config-vlan)# exit
```

---

### 🔍 Verify VLAN Creation

```Shell
Switch# show vlan brief
```

✅ VLAN 30 and VLAN 50 created successfully

---

## 🔌 Step 3: Configure Access Ports

Access ports are used to connect **end devices** (PC, IP Camera, Printer).

---

### ✅ Configure Single Port (Gi1/0/1 → VLAN 30)

```Shell
Switch(config)# interface gigabitEthernet 1/0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 30
Switch(config-if)# exit
```

---

### ✅ Configure Multiple Ports (Gi1/0/2–4 → VLAN 30)

```Shell
Switch(config)# interface range gigabitEthernet 1/0/2-4
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 30
Switch(config-if-range)# exit
```

---

### ✅ Assign VLAN 50 to Ports 9–12

```Shell
Switch(config)# interface range gigabitEthernet 1/0/9-12
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 50
Switch(config-if-range)# exit
```

---

### 🔍 Verify Port Assignment

```Shell
Switch# show vlan brief
```

✅ Ports assigned:

- VLAN 30 → Ports 1–4
- VLAN 50 → Ports 9–12

---

## 🔄 Access Port vs Trunk Port

### 🔹 Access Port

- Used to connect **end devices**
- Carries **single VLAN traffic only**
- Example: PC, CCTV Camera

---

### 🔹 Trunk Port

- Used to connect **switch-to-switch or router**
- Carries **multiple VLAN traffic**
- Uses **802.1Q tagging**

---

## 🔗 Step 4: Configure Trunk Port

### ✅ Configure Gi1/0/24 as Trunk

```Shell
Switch(config)# interface gigabitEthernet 1/0/24
Switch(config-if)# switchport mode trunk
Switch(config-if)# switchport trunk allowed vlan 1,30,50
Switch(config-if)# exit
```

✅ Trunk port configured successfully
