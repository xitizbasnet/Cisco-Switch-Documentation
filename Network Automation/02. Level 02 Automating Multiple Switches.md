# 🟡 Level 2: Automating Multiple Cisco Switches

Now we scale automation to **multiple devices**.

---

## 🧾 Step 1: Define Multiple Switches

We store all devices in a Python list:

```python
switches = [
    {
        "device_type": "cisco_ios",
        "host": "192.168.99.1",
        "username": "admin",
        "password": "password"
    },
    {
        "device_type": "cisco_ios",
        "host": "192.168.99.2",
        "username": "admin",
        "password": "password"
    }
]
```

### 📌 Key Idea

Each dictionary = one switch
The list = your entire network scope

---

## ⚙️ Step 2: Define Configuration Commands

```python
config_commands = [
    "vlan 10",
    "name IT",
    "vlan 20",
    "name HR",
    "vlan 30",
    "name FINANCE"
]
```

### 📌 Purpose

These commands will be executed on **every switch** automatically.

---

## 🔄 Step 3: Loop Through All Switches

```python
for switch in switches:
    net_connect = ConnectHandler(**switch)

    print(f"Connected to {switch['host']}")
    print("Applying configuration...")

    net_connect.send_config_set(config_commands)

    print("Configuration applied successfully!")

    output = net_connect.send_command("show vlan brief")
    print("\n🔍 Verification Output:")
    print(output)

    net_connect.send_command("write memory")

    net_connect.disconnect()

    print("Connection closed and configuration saved.\n")
```

---

## 🧠 How the Loop Works

```python
for switch in switches:
```

* Iteration 1 → Switch 192.168.99.1
* Iteration 2 → Switch 192.168.99.2
* Continues until all devices are processed

### 📌 Result

The same configuration is applied **consistently across all switches**.

# ✅ Final Production Script

```python
from netmiko import ConnectHandler

switches = [
    {
        "device_type": "cisco_ios",
        "host": "192.168.99.1",
        "username": "admin",
        "password": "password"
    },
    {
        "device_type": "cisco_ios",
        "host": "192.168.99.2",
        "username": "admin",
        "password": "password"
    }
]

config_commands = [
    "vlan 10",
    "name IT",
    "vlan 20",
    "name HR",
    "vlan 30",
    "name FINANCE"
]

for switch in switches:
    net_connect = ConnectHandler(**switch)

    print(f"✅ Connected to {switch['host']}")
    print("⚙️ Applying configuration...")

    net_connect.send_config_set(config_commands)

    print("✅ Configuration applied successfully!")

    output = net_connect.send_command("show vlan brief")
    print("\n🔍 Verification Output:")
    print(output)

    net_connect.send_command("write memory")

    net_connect.disconnect()

    print("✅ Connection closed and configuration saved.\n")
```
