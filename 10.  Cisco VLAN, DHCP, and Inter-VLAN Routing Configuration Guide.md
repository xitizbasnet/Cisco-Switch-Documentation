# 📘 Cisco VLAN, DHCP, and Inter-VLAN Routing Configuration Guide

### 🔧 Assign IPs to VLANs | Configure Multiple DHCP Pools | Enable Inter-VLAN Routing

---

## 📌 Overview

Do you know how to assign IP addresses to a VLAN and how to create multiple DHCP servers in your Cisco switch? If not, then this guide is for you.

In this documentation, I will show you:

- How to assign an IP address to VLANs in a Cisco switch
- How to create multiple DHCP servers
- How to exclude specific IP addresses from DHCP pools
- How to enable inter-VLAN routing on a Cisco switch

---

## 🔐 Accessing the Cisco Switch

```Shell
Switch> enable
Switch#
```

---

## 🔍 Step 1: Verify Existing VLAN Configuration

```Shell
Switch# show vlan brief
```

### ✅ Existing VLANs

- VLAN 30 → Name: Data
- VLAN 50 → Name: CCTV

### 🔌 Port Assignment

- Ports 1–4 → VLAN 30
- Ports 9–12 → VLAN 50

---

## 🌐 Step 2: Assign IP Address to VLAN Interfaces (SVI)

To create DHCP servers, we must first assign gateway IP addresses to VLAN interfaces.

### ⚙️ Enter Global Configuration Mode

```Shell
Switch# configure terminal
```

---

### 🔧 Configure VLAN 30 Interface

```Shell
Switch(config)# interface vlan 30
Switch(config-if)# ip address 10.0.30.254 255.255.255.0
Switch(config-if)# no shutdown
Switch(config-if)# exit
```

✅ VLAN 30 gateway configured

---

### 🔧 Configure VLAN 50 Interface

```Shell
Switch(config)# interface vlan 50
Switch(config-if)# ip address 10.0.50.254 255.255.255.0
Switch(config-if)# no shutdown
Switch(config-if)# exit
```

✅ VLAN 50 gateway configured

> 💡 Note: You can use `.1`, `.254`, or any valid IP as gateway.

---

## 🖧 Step 3: Configure DHCP Server (Multiple Pools)

---

### ✅ DHCP Pool for VLAN 30

```Shell
Switch(config)# ip dhcp pool VLAN30
Switch(dhcp-config)# network 10.0.30.0 255.255.255.0
Switch(dhcp-config)# default-router 10.0.30.254
Switch(dhcp-config)# dns-server 8.8.8.8
Switch(dhcp-config)# exit
```

---

### ✅ DHCP Pool for VLAN 50

```Shell
Switch(config)# ip dhcp pool VLAN50
Switch(dhcp-config)# network 10.0.50.0 255.255.255.0
Switch(dhcp-config)# default-router 10.0.50.254
Switch(dhcp-config)# dns-server 8.8.8.8
Switch(dhcp-config)# exit
```

✅ Two DHCP servers successfully created

---

## 🚫 Step 4: Exclude IP Addresses from DHCP

To reserve IPs (e.g., for servers or static devices):

### 🔒 VLAN 30 Exclusion

```Shell
Switch(config)# ip dhcp excluded-address 10.0.30.1 10.0.30.10
```

✅ First 10 IPs reserved

---

### 🔒 VLAN 50 Exclusion

```Shell
Switch(config)# ip dhcp excluded-address 10.0.50.1 10.0.50.20
```

✅ First 20 IPs reserved

---

## 🔁 Step 5: Enable Inter-VLAN Routing

```Shell
Switch(config)# ip routing
```

✅ Enables communication between VLANs

> ⚠️ Works only on **Layer 3 switches**. If not supported, use a router.

---

## 💾 Step 6: Save Configuration

```Shell
Switch# write memory
```
